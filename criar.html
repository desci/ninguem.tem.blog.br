Octopress

Octopress é uma ferramenta para providenciar blogs para hackers. É necessário ter conhecimento básico de markdown e git para utilizar esta ferramenta. Ou então ter conta em algum serviço de hospedagem git, como por exemplo Gitorious ou GitHub.
